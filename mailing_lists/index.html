Apache Traffic Control - Mailing Lists

The following lists are the preferred methods of communication for the Traffic Control Project. Archives are available at Apache Mail Archives.

Subscription instructions and further Apache Foundation-wide mailing lists can be found at Apache Mailing Lists

Name Address Description
Users users@trafficcontrol.apache.org General list for questions and answers, suitable for all users
Development dev@trafficcontrol.apache.org Development proposals, discussion and release voting
Security security@trafficcontrol.apache.org Reporting of security vulnerabilities
Commits commits@trafficcontrol.apache.org Archive mailer for all Traffic Control commits
Issues issues@trafficcontrol.apache.org Archive mailer for all Traffic Control Github Issues/PullRequests
